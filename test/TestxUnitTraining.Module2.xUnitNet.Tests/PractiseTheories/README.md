# Prácticas propuestas para el módulo - Parte I

* Crea pruebas  para `ColorFactory` de modo que el código del proyecto esté probado en su totalidad.
* Crea un atributo Custom que utilice el fichero .json que está en la raíz, para ser utilizado con la teoría del método GetColorComposition. No es necesario que crees un atributo genérico que sirva para cualquier fichero .json, puede ser a medida para este caso concreto.

 > Recuerda que, para que las pruebas sean útiles, no solo hay que probar el resultado, sino todos los caminos lógicos que pueda seguir el código. Esto incluye las excepciones y los retornos nulos.
